# Readme

## Mac OS

Run `brew unlink imagemagick`

Run `brew install imagemagick@6 -s && brew link imagemagick@6 --force`

`gem install rmagick`

`bundle install`

## Linux

`sudo apt-get install libmagickwand-dev`

## Possible colors
